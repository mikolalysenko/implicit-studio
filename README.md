implicit-studio
===============
A basic in browser editor for rendering implicit functions.

Try it out right now (requires WebGL):

http://mikolalysenko.github.io/implicit-studio/


## Controls

* Left click/A to rotate
* Right click/S to pan
* Middle click/D to zoom

The left editor pane defines the implicit function

The right pane defines the color function
